# Extra- Saída pelo PMOD (Osciloscópio)
Disponibiliza o sinal modulado em uma saída VGA da
placa, permitindo visualizar a forma de onda em um monitor
através de cores pulsantes

## Estrutura desta pasta
```
vga/
|-- src/                             Código VHDL adaptado com saída no VGA
|-- constraints/                     Constraint com mapeamento do pino VGA
`-- docs/                            
```

## Equipamentos necessários:
- Placa FPGA Basys 3
- Cabo VGA
- Monitor
## Como reproduzir
- Siga o procedimento de gravação na placa da mesma forma que os outros
exemplos.
- Conecte o VGA na placa FPGA e no Monitor
